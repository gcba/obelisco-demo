Aplicaciones

Mapa Interactivo de Buenos Aires

Brinda información geográfica de interés general a toda la ciudadanía, presenta los servicios que ofrece la Ciudad en los diferentes barrios, además de herramientas para la localización y búsqueda de recorridos en transporte público, auto, bici o a pie.

Mapa Interactivo de Buenos Aires

BA Cómo Llego

Ayuda a los usuarios a moverse por la ciudad en transporte público, auto, bici o a pie, ofreciendo las mejores rutas, indicadores de tiempo y costo. Integra información de tiempo real de subtes, trenes y colectivos para conocer cuánto falta para que llegue el próximo transporte. Está disponible para iOS y Android en español y en inglés.

BA Cómo Llego (Android) BA Cómo Llego (iOS)

Normalizador de Bases de Direcciones

Una solución amigable y efectiva para la normalización automática de direcciones en conjuntos de datos. Incluye las herramientas necesarias para corregir interactivamente los registros erróneos y la posibilidad de geocodificar de forma rápida, sencilla y precisa.

Requerimientos: Windows XP o superior. Conexión a Internet

Manual Descargar

Api Manager

Interfaz de acceso y exploración de los servicios Web disponibilizados para la ejecución de procesos geoespaciales desde otras aplicaciones. Los servicios Web brindan acceso a las soluciones de normalización, geocodificación, consulta de datos útiles y búsquedas personalizadas, entre otras.

Componentes para Desarrolladores

USIG cuenta con diferentes componentes para desarrolladores que permite recrear productos y contribuir con la comunidad a través de repositorios de código abierto.

Repositorios:

Github NPM

Librería de Mapa interactivo

Permite crear de manera simple un mapa interactivo de la Ciudad de Buenos Aires y alrededores.

Github NPM

Librería de Recorridos

Servicio para consultar recorridos en transporte público o bicicleta dentro de la ciudad de Buenos Aires

NPM

Librería de Gestor de Callejero

Esta librería cuenta con los datos de todas las calles de la Ciudad de Buenos Aires. Se puede utilizar tanto en el cliente como en Node.js.

NPM

Librería de Autocompleter

Esta librería nos permite disponer dentro de nuestro desarrollos un componentes para autocompletar la búsqueda de direcciones.

NPM

Librería de Clientes de Normalizador de direcciones Python - AMBA

Este componentes está desarrollado en python y nos permite de manera simple normalizar direcciones de AMBA y de CABA.

Github PyPI
